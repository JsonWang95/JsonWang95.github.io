Music Rec Full Stack Webapp

Background

The idea of this side project is that I want to build a webapp where users can have more interaction when listening to music. Currently, Spotify has the feature to show what music you’re listening to to your friends. Howevert, I think it’s not good enough for users to actually use it despite being a great idea. Thus, this project servers as a practice opportunity to build a full stack webapp where I aim to improve the interaction mechanism between users who want to recommend music to friends or even starngers on the internet.

Frontend npx create-react-app frontend command vulneribility issues

When initializing the frontend directory, after running the npx command and running npm-audit, the console kept giving me vulnerability issues, including severe ones where I had to fix right away. After trying to fix them one by one manually, the issues persisted. After searching a little more, I found that npm auditis actually broken for front-end tooling by design, read here: https://github.com/facebook/create-react-app/issues/11174

After the following the steps suggested, the vulnerability issues were all gone.
